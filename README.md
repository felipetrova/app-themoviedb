## :information_source: How To Use
```bash

# Clone this repository
$ git clone https://github.com/felipetrova/app-themoviedb

# Go into the repository
$ cd navigate/to/app-themoviedb

# Install dependencies
$ npm install

# Run the app
$ ng serve

```
